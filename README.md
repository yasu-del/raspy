# Raspy

このリポジトリは、Raspberry Piに接続されたI2Cデバイス（BNO055 9軸慣性センサ）からデータをリアルタイムに取得するためのプロジェクトです。

## 構成ファイル

- [bno055_reader.py](file:///c:/Users/yasu/Documents/GitHub/raspy/bno055_reader.py): Raspberry PiのI2Cバスから直接BNO055のセンサデータ（オイラー角、クォータニオン、温度、キャリブレーション状態）を読み出すPythonスクリプト。

## 実行方法

ラズパイ側のターミナルで以下のコマンドを実行します：

```bash
python3 bno055_reader.py
```

### 接続構成と前提条件

- **対象センサ**: BNO055 (I2Cアドレス: `0x28`)
- **依存関係**: 標準ライブラリの `smbus` (または `smbus2`) がラズパイにインストールされていること。

### 特徴

BNO055とのI2C通信において発生しやすい「クロックストレッチング（通信同期ズレ）」によるデータ化け（温度や角度の瞬間的なバグ値）に対処するため、レジスタの読み込みルーチンに微小なディレイを挟み、再試行処理を行うことで高い動作安定性を確保しています。
